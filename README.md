## Getting Started

### Prerequisites
- GCC or Clang compiler
- Make (optional, for building)

### Compilation
To compile C files:
```bash
gcc -o output_file source_file.c
```

To compile C++ files:
```bash
g++ -o output_file source_file.cpp
```
